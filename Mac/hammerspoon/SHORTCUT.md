# Keyboard Shortcuts

## Windows management

| Shortcut                      | Description                  | Source      |
|-------------------------------|------------------------------|-------------|
| ⌃ + ⌥ + ⌘ + →                 | Focus next display           | Hammerspoon |
| ⌃ + ⌥ + ⌘ + ←                 | Focus previous display       | Hammerspoon |
| ⌃ + ⌥ + ⌘ + U                 | Move window to top left      | Hammerspoon |
| ⌃ + ⌥ + ⌘ + I                 | Move window to top right     | Hammerspoon |
| ⌃ + ⌥ + ⌘ + J                 | Move window to bottom left   | Hammerspoon |
| ⌃ + ⌥ + ⌘ + K                 | Move window to bottom right  | Hammerspoon |
| ⌃ + ⌥ + ⌘ + ò                 | Move window to left half     | Hammerspoon |
| ⌃ + ⌥ + ⌘ + ù                 | Move window to right half    | Hammerspoon |
| ⌃ + ⌥ + ⌘ + è                 | Move window to top half      | Hammerspoon |
| ⌃ + ⌥ + ⌘ + à                 | Move window to bottom half   | Hammerspoon |
| ⌃ + ⌥ + ⌘ + ↩                 | Maximize window              | Hammerspoon |
| ⌃ + ⌥ + ⌘ + PgDn              | Move window to next screen   | Hammerspoon |
| ⌃ + ⌥ + ⌘ + PgUp              | Move window to prev screen   | Hammerspoon |
| ⌘ + ⇥                          | Switch to next app                   | macOS Default  |
| ⌘ + H                          | Hide current app                     | macOS Default  |
| ⌘ + M                          | Minimize window                      | macOS Default  |
| ⌘ + W                          | Close window/tab                     | macOS Default  |
| ⌘ + Q                          | Quit app                             | macOS Default  |
| ⌘ + Space                      | Spotlight search                     | macOS Default  |
| ⌃ + ↑                          | Mission Control                      | macOS Default  |
| ⌃ + ↓                          | App Exposé                           | macOS Default  |
| ⌃ + ←/→                        | Switch desktop (Space)               | macOS Default  |
| ⌘ + ⇧ + 3                      | Screenshot (full screen)             | macOS Default  |
| ⌘ + ⇧ + 4                      | Screenshot (selection)               | macOS Default  |
